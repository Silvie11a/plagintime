# Insert Timestamp

**Автор**: Бугаева Элина Константиновна, группа M3108  
**Название плагина**: Insert Timestamp  
**Описание**: Плагин для Visual Studio Code, который вставляет текущую дату и время в формате `ГГГГ-ММ-ДД ЧЧ:ММ:СС` в текущую позицию курсора в редакторе.

## Возможности
- Вызов команды через палитру команд (`Ctrl+Shift+P` → "Insert Timestamp").
- Автоматическая вставка отметки времени без выделения текста.
- Работает в любом текстовом файле (`.txt`, `.py`, `.js`, `.md` и т.д.).

## Установка (локально)
1. Склонируйте репозиторий.
2. Откройте папку в VS Code.
3. Нажмите F5 для запуска в режиме отладки.
4. В новом окне вызовите команду через палитру.

Для публикации: используйте `vsce package` и установите `.vsix` файл.
